# TasteBite-Sales-Performance-Analysis
This project involved developing a comprehensive Power BI dashboard for TasteBite, a fast-moving consumer goods (FMCG) organization, to transform sales data into actionable business intelligence that supports strategic decision-making, financial evaluation, and operational improvement.
